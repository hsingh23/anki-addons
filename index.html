anki-addons

A collection of Anki2 addons in different states of bugginess and usefulness.

Backdoor

Status: working
Use one specific password to access accounts “locked” with a password.

Add note id

Status: working
The first field of cards should be unique. To make sure this is the case, this add-on adds a unique number to fields called `Note ID`.

Dvorak keys

Status: working
Use keys easy to use for user of the dvorak simplified keyboard to answer a question.

Anki 1.2 shortcuts

Status: working
Re-activate F5, Ctrl+f and Ctrl+w shortcuts.

Compare to kana

Status: working
When typing in Japanese readings, remove the kanji bit from the correct text, so that the red and green coloring of the answer works correctly.

Deck name in title

Status: working
Show the name of the deck you are currently learning in the window title.

Local CSS

Adapt card styles to time and place

Status: working
Add styling from a local CSS file. With this, the card style can be adapted to time (night colors) and place (small fonts on small screens, big fonts on big screens, ...).

Nachschlagen

Status: working
Menu to look up words at a few web sites. Most useful for speakers of German learning Japanese

Scalar

Status: working
Compare a typed in number with the correct answer as a number, not as a string of digits. Color it yellow when it is close to the right number.

Colorful toolbar

Status: working
Replace the top menu bar with the “Decks Study Add Browse” text buttons and the two rather drab sync and statistics buttons with a colorful icon toolbar like the one in Anki 1.

Modified furigana

Status: for desktop use only
Improved way to parse kanji and readings, a way to render kanji the kana and a few other display hacks.

Metric time

Status: personal hobby horse
Shows times in some spots as days or years.

Swiss locale

Status: one-off
An add-on to make my German geography deck work.

Quick replay

Status: hackish
Play sound files with dedicated programs, bypassing the mplayer play queue. Like this, when you hit replay, the replay starts at once.

Kanji stroke color

Status: hackish, desktop only
A quick hack add-on to show (colored) stroke order diagrams.

Show colored stroke order diagrams

Status: early development
Another take on displaying the KanjiVG stroke order diagrams with color. Not much more than an idea how to go about it at the moment.

Furikanji

Status: obsolete
Render kanji above the kana

Play with mpg123

Status: obsolete
A quick hack add-on to play mp3 files with mpg123 or mpg321

Play with sox

Status: useless
The idea was to make flac and ogg audio files usable on Windows systems.

faenza-icons

Status: abandoned
Replace the sync and statistics icons

Download audio

Status: early development
The goal is to automatically download pronunciations from several sites.

Dehashilator

Status: breaks collections
Rename media files with names that look like MD5 hashes.

Classy correct

Status: abandoned
An abandoned effort to add a css class to typed answers.

No password

Status: obsolete
Open accounts with a password set without entering the password.
